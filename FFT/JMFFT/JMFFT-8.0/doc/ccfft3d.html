NOM

               dimensions (3D) complexe-complexe.

IMPLEMENTATION

     Ce sous-programme ťmule le sous-programme de mÍme nom de la SCILIB de
     CRAY. Tous les arguments rťels ou complexes doivent Ítre dťclarťs
     en double prťcision.

DESCRIPTION

     CCFFT3D calcule la FFT du tableau complexe X et retourne le rťsultat dans
     le tableau complexe Y.

     Soient deux tabeaux X et Y dimensionnťs de la faÁon suivante :

	  COMPLEX(KIND=8), DIMENSION(0:n1-1, 0:n2-1, 0:n3-1) :: X
	  COMPLEX(KIND=8), DIMENSION(0:n1-1, 0:n2-1, 0:n3-1) :: Y

     CCFFT3D applique la formule suivante :

	                  n1-1 n2-1 n3-1
    Y(k1,k2,k3)	= scale * Sum  Sum  Sum [X(j1,j2,j3)*w1**(j1*k1)*w2**(j2*k2)*w3**(j3*k3)]
	                  j1=0 j2=0 j3=0

    pour k1 = 0,	..., n1	- 1,
	 k2 = 0,	..., n2	- 1,
	 k3 = 0,	..., n3	- 1,

    oý,
	w1 = exp(isign*2*pi*i/n1),
	w2 = exp(isign*2*pi*i/n2),
	w3 = exp(isign*2*pi*i/n3),
	i  = + sqrt(-1)
	pi = 3.14159...
	isign =	+1 ou -1

     En gťnťral, si une FFT est appliquťe avec des valeurs particuliŤres de
     isign et scale, alors son inverse est calculťe avec les valeurs -isign
     et 1/(n1*n2*n3*scale).

     En particulier, avec les valeurs isign = +1 et scale = 1.0, la FFT
     inverse se calcule en prenant isign = -1 et scale = 1/(n1*n2*n3).

	     Nombre de transformťe dans la premiŤre dimension.

	     Tableau contenant la valeurs des ťlťments ŗ transformer.

     Exemple 1 : initialise le tableau TABLE dans le but d'appliquer
     ultťrieurement une FFT de dimensions (128,128,128). Dans ce cas, seuls les
     arguments ISIGN, N1, N2, N3 et TABLE sont utilisťs.

     ťlťments de chaque dimension. Le tableau TABLE est supposť Ítre initialisť.

	   REAL(KIND=8), DIMENSION(100 + 2*(128 + 128 + 128)) :: TABLE
	   REAL(KIND=8), DIMENSION(512*128)                   :: WORK
	   ...

                        1, 1, TABLE, WORK, 0)
	   CALL	CCFFT3D(1, 128,	128, 128, 1.d0, X, 129, 129, Y, &
                        129, 129, TABLE, WORK, 0)

     Exemple 3 : ici nous poursuivons l'exemple 2 en effectuant la FFT inverse
     de Y et en enregistrant le rťsultat dans X.  Le facteur d'ťchelle est ici
     ťgale ŗ 1/(128*128*128). Nous supposons que le tableau TABLE ait ťtť
     initialisť auparavant.

	                X, 129, 129, TABLE, WORK, 0)

     Exemple 4:	 nous effectuons ici un calcul analogue ŗ celui de l'exemple 2
     en supposant toutefois que les indices des tableaux X et Y dťmarrent ŗ 1
     et non plus ŗ 0.

	   ...

	                DUMMY, 1, 1, TABLE, WORK, 0)
	   CALL	CCFFT3D(1, 128,	128, 128, 1.d0, X, 129, 129,  &
	                Y, 129, 129, TABLE, WORK, 0)

     Exemple 5 :	calcul semblable ŗ l'exemple 4 sauf qu'ici nous utilisons X ŗ

	                X, 129, 129, TABLE, WORK, 0)

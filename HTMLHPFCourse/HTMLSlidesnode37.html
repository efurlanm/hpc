Relational Operators

The following relational operators deliver a LOGICAL result when combined with numeric operands,

For example, 

    bool = i .GT. j
    boule = i > j
    IF (i .EQ. j) c = D
    WHERE (a(:) /= 0) a(:) = 1/a(:)

Assignment

Assignment is defined between all expressions of the same type:

Examples, 

    a = b
    c = SIN(.7)*12.7  ! SIN in radians
    name = initials//surname
    bool = (a.EQ.b.OR.c.NE.d)

The LHS is an object and the RHS is an expression.

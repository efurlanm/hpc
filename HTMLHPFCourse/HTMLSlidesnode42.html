Control Flow

Control constructs allow the normal sequential order of execution to be changed. Fortran 90 supports:

  • conditional execution statements and constructs, (IF ... and IF ... THEN ... ELSE ... END IF);
  • loops, (DO ... END DO);
  • multi-way choice construct, (SELECT CASE);
and less importantly,
  • unconditional jump statements, (GOTO);
  • I/O exception branching, (ERR=);

MIMD Example

Multiple Instructions Multiple Data, 

    REAL FUNCTION F(x,i) ! PURE
     IMPLICIT NONE
     REAL, INTENT(IN) :: x    ! element
     INTEGER, INTENT(IN) :: i ! index
      IF (x > 0.0) THEN
       F = x*x
      ELSEIF (i==1 .OR. i==n) THEN
       F = 0.0
      ELSE
       F = x
      END IF
    END FUNCTION F

  • used as alternative to WHERE or FORALL.

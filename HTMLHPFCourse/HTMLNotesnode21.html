Undesirable Features

The following features are not marked as obsolescent yet can, and indeed should be, expressed using new syntax:


  • Fixed Source Form
  • Implicit Declaration of Variables
  • COMMON Blocks
  • Assumed Size Arrays
  • EQUIVALENCE Statement
  • ENTRY Statement

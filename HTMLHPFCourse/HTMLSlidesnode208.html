Do-loops and Forall Statements

Take care, FORALL semantics are different to DO-loop semantics, 

    DO i = 2, n-1
     a(i) = a(i-1) + a(i) + a(i+1)
    END DO

is different to, 

    FORALL (i=2:n-1) &
     a(i) = a(i-1) + a(i) + a(i+1)

which is the same as, 

     a(2:n-1) = a(1:n-2) + a(2:n-1) + a(3:n)

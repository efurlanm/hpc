Subroutine Syntax

Syntax of a (non-recursive) subroutine declaration: 

 
 SUBROUTINE    procname   [ (   dummy args   ) ]


 		    declaration of dummy args   


 		    declaration of local objects   


 		 		 ...


 		    executable stmts   


  [ CONTAINS


 		    internal procedure definitions    ]


 END [ SUBROUTINE [   procname    ] ]

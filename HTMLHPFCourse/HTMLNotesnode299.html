Features Demoted to be Extensions

  • REDISTRIBUTE and REALIGN are deemed too inefficient to be in the main language,
  • the DYNAMIC attribute which is related to the above,

In addition to these the rules have been changed regarding the INHERIT directive and explicit intterfaces and now required if remapping occurs across a procedure boundary. It is also not permitted to map sequential arrays (eg, assumed-size arrays).

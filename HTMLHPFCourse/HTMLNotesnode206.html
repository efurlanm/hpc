Simple 2D Alignment Example

Given, 

    REAL, DIMENSION(10,10) :: A, B
!HPF$ ALIGN A(:,:) WITH B(:,:)

This is much the same as the previous example except that we are aligning 2D arrays. Each array is aligned dimension by dimension. The net effect of this alignment is that A(i,j) and B(i,j) reside on the same processor. Effectively, this says: i, j, elements A(i,j) and B(i,j) are local.

The following align statement is equivalent but does not imply shape conformance: 

!HPF$ ALIGN A(i,j) WITH B(i,j)

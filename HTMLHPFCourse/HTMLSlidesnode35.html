Intrinsic Character Operations - Substrings

Consider, 

   CHARACTER(LEN=*), PARAMETER :: string = "abcdefgh"

substrings can be taken,

  • string is `abcdefgh'
  • string(1:1) is `a' (1st letter)
  • string(2:4) is `bcd' (2nd, 3rd and 4th letters).
  • string(1) is an error.
